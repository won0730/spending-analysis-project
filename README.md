# 소비 패턴 분석 프로젝트

python과 SQL을 활용하여 소비 데이터를 분석하고 시각화하는 프로그램

## 사용 언어 및 라이브러리
- **Language**: Python
- **Database**: MySQL
- **Library**: Pandas, Matplotlib, Seaborn

## 주요 기능
- **카테고리별 지출 분석**: 파이 차트를 통한 카테고리별 소비 비중 확인
- **주별 지출 추이**: 선 그래프를 통한 주별 소비 시각화
- **일별 소비 히트맵**: 한 달간의 일별 지출 강도를 히트맵 형태로 확인
- **소비 목표 관리**: 목표 금액 초과 시 경고 및 잔액 계산
- **소비 예측**: 과거 데이터를 이용한 앞으로의 소비 예측
